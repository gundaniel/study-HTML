이벤트(Event)

        이벤트(Event):동작, 행위
        ->브라우저에서읟 동작, 행위: click, keydown, keyup, mouseover, submit,change ...

        이벤트 리스너(Event Listner)
        ->이벤트가 발생하는 것을 대기하고 있다가
        이벤트가 발생하는 거싱 감지 되면 연결된 기능을 수행하는 것
        ex) onclick, onkeyup,onchange, onsubmit...(on이벤트명)
        onclick="함수명()"

        이벤트 핸들러 (Event Handler)
        -> 이벤트 리스너에 연결된 기능으로
        이벤트 발생을 수행하고자 하는 내용을 작성해둔 함수(function)

인라인 이벤트 모집

        요소 내부에 이벤트를 작성하는 바업븡로 
        on이벤트명 - 함수명 () 형태로 작성함
    
        색상명 클릭
  • Green
  • Orange
  • Purple
넌센스 퀴즈
        1. 질문과 정답에 해당하는 이미지를 다운받는다. 
        2. 질문 이미지를 클릭하면 정답 이미지를 출력하면 된다.
           이때 요소.src="경로"를 명시해주면 이미지를 변경할 수 있다.

고전 이벤트 모델

        요소가 가지고 있는 이벤트 속성(이벤트 리스너)에 이벤트 핸들러를 연결하는 방법으로

        인라인 모델처럼 HTML 요소에 JS 코드가 포함되는 것이 아닌
        script 내에서 이벤트 관련코드를 작성할 수 있다.

노력은 결고 배신하지 않는다.

표준이벤트 모델(addEventListner)

        - W3C(HTML, CSS, JS 웹표준 재정 단체)에서 
        공식적으로 지정한 이벤트 모델(이벤트 동작 지정 방법)

        - 한 요소에 여러 이벤트 핸들러를 설정할 수 있다.
